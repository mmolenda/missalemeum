iCalendar

1962 liturgical calendar is available in iCalendar format.

https://www.missalemeum.com/en/api/v3/icalendar

It can be loaded into any calendar application, such as Google Calendar (manual), or Microsoft Outlook. The link contains all Sundays and major feasts. In the description of each calendar event there is a link to the respective proper on the website.

Additionally, you can specify what rank of feasts are included in the ical feed by adding the optional paramater to the URL in the last field.

First class feast feed

https://www.missalemeum.com/en/api/v3/icalendar/1

First and second class feast feed

https://www.missalemeum.com/en/api/v3/icalendar/2

First, second, and third class feast feed

https://www.missalemeum.com/en/api/v3/icalendar/3

All feasts feed:

https://www.missalemeum.com/en/api/v3/icalendar/4
